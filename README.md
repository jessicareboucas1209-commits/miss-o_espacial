# missao_espacial
🚀 Projeto: Missão Espacial - Engine de Lógica em Python
Este repositório contém o desenvolvimento de uma aplicação de console em Python focada na implementação de algoritmos de tomada de decisão e processamento de dados técnicos. O projeto foi estruturado para demonstrar conceitos fundamentais de Desenvolvimento de Sistemas.


💻 Objetivos Técnicos
Lógica de Programação: Implementação de estruturas condicionais complexas para filtragem de dados.

Manipulação de Tipos: Gerenciamento de variáveis flutuantes, inteiras e strings com tratamento de entrada de usuário.

Arquitetura de Software: Organização de script único para execução rápida em ambiente CLI (Command Line Interface).


🛠️ Funcionalidades Implementadas
Módulo de Cálculo: Processamento de fórmulas matemáticas aplicadas a variáveis de entrada.

Sistema de Triagem Lógica: Algoritmo que classifica dados com base em parâmetros pré-definidos.

Interface de Terminal: Fluxo de interação limpo para entrada e exibição de resultados em tempo real.


🚀 Tecnologias e Ferramentas
Linguagem: Python 3.10+

Versionamento: Git

Plataforma de Hospedagem: GitHub

Ambiente de Desenvolvimento: Visual Studio Code
